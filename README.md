# Neovim (Lazynvim) (Arch Linux)

## Need to install those things to make null-ls (code formatter) works

**Prettier (For web)**

```
yay -S prettier
```

**Stylua (For lua)**

```
yay -S stylua
```

**Uncrustify (For C, C++, C#, Java)**

```
yay -S uncrustify
```

**_Yapf (For Python)_**

```
yay -S yapf
```
